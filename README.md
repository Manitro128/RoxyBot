# RoxyBot
Bot discord termux


# FOR TERMUX
```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
git clone https://github.com/Nobuyaki/RoxyBot.git
cd RoxyBot
npm install
node . || node index.js
```

#### config.json
```bash
["./data/config.json"](https://github.com/Nobuyaki/RoxyBot/blob/main/data/config.json)
{
  "token": "your-bot-token-here",
  "prefix": "r!"
}
```
